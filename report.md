# Phishing Email Investigation Report

## 1. Overview

This project simulates a phishing attack and investigates its indicators.

## 2. Environment

* Kali Linux (Attacker)
* Windows 11 (Victim)

## 3. Email Details

* Subject: Critical Security Update Required
* Sender: it.adminsupport2@gmail.com

## 4. Analysis

### 4.1 Sender Analysis

The sender email appears to impersonate a legitimate service by using a similar name.

### 4.2 URL Analysis

The URL contains a private IP address, which is unusual for legitimate services.

### 4.3 VirusTotal Analysis

The URL was checked and found clean, likely due to lack of public reputation.

### 4.4 Header Analysis

Email headers were reviewed to verify sender path and authenticity.

## 5. Phishing Indicators

* Spoofed sender email
* Suspicious internal IP address
* Urgent message tone
* Generic greeting

## 6. Conclusion

The email is confirmed as a phishing attempt.
