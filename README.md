# Phishing Email Investigation Lab

## 📌 Objective

To simulate and analyze a phishing email attack in a controlled lab environment.

## 🛠 Lab Setup

* Attacker: Kali Linux
* Victim: Windows 11
* Email Service: Gmail SMTP

## ⚙️ Tools Used

* Python (smtplib)
* VirusTotal
* Web Browser

## 🪜 Process Overview

1. Created attacker and victim email accounts
2. Generated Gmail app password
3. Sent phishing email using Python script
4. Received email on Windows 11
5. Analyzed email content and link
6. Extracted indicators of compromise (IOCs)

## 🚨 IOCs

* Email: it.adminsupport2.@gmail.com
* URL: http://192.168.0.200/login
* IP: 192.168.0.200

## ✅ Conclusion

The email was identified as a phishing attempt based on spoofed sender, suspicious link, and social engineering techniques.
